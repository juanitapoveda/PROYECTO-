# DESCRIPCION DEL PROYECTO:
Es un sistema basico de gestion de peliculas, el cual incluye diferentes funcionalidades para que el usuario opere e interactue con el, como ejemplo, reagendamiento de peliculas, encontrar peliculas, buscar peliculas de estreno, entre otras opciones; desarrollado en python con apoyo de herramientas como condicionales, bucles, diccionarios, listas y funciones que permiten el funcionamiento de este.
